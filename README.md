# Prodigy DS task 05 
🚗 Traffic Accident Analysis & Hotspot Visualization


🔍 Project Summary
Conducted an in-depth analysis of traffic accident data to uncover patterns related to road conditions, weather, and time of day. This project aims to enhance road safety by identifying key contributing factors and visualizing accident hotspots, enabling better urban planning, policy-making, and driver awareness initiatives.

🧰 Tools & Technologies
Languages & Libraries: Python, Pandas, Seaborn, Matplotlib, Plotly, Folium

Techniques: Time series analysis, correlation mapping, geospatial visualization

Data Sources: Public accident datasets (e.g., U.S. DOT, UK GOV, Kaggle)

📊 Process Breakdown
1. Data Cleaning & Preprocessing
Parsed and cleaned large datasets containing timestamp, location, road, and weather info

Handled missing values, standardized date-time formats, and categorized weather/road types

2. Pattern Analysis
Analyzed frequency of accidents by hour, day, and season

Investigated correlations between weather conditions (e.g., fog, rain, snow) and accident rates

Identified accident patterns across different road types and lighting conditions

3. Visualization
Created heatmaps and scatter plots to highlight high-risk zones

Used geospatial tools (Folium/Plotly) to create interactive accident hotspot maps

Built dashboards showing accident counts and severity distribution by category

💡 Key Insights
Accident likelihood peaks during rush hours and under wet or low-light conditions

Intersections and urban zones with poor lighting had higher incident rates

Weather factors like fog and rain significantly increased accident severity

🚀 What I Learned
How to work with geospatial and time-based datasets for real-world safety analysis

The power of data visualization in revealing hidden patterns and high-risk zones

How insights from data can guide infrastructure improvements and public safety campaigns
